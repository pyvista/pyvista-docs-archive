Text3D#

Text3D(string, depth=0.5)[source]#

Create 3D text from a string.

Parameters:
stringstr

String to generate 3D text from.

depthfloat, default: 0.5

Depth of the text.

Returns:
pyvista.PolyData

3D text mesh.

Examples

>>> import pyvista
>>> text_mesh = pyvista.Text3D('PyVista')
>>> text_mesh.plot(cpos='xy')
